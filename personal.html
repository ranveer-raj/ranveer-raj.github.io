Relevant Computer Science Courses Description
Programming and Data Structures (Theory + Lab) Theory: Introduction to fundamental programming concepts and data structures such as arrays, linked lists, stacks, and queues. Understanding of algorithms, time complexity, and space complexity.

Lab: Practical application of programming skills with hands-on exercises focusing on data structure implementation and problem-solving.
Algorithms-I (Theory + Lab) Theory: In-depth study of fundamental algorithms, algorithmic design paradigms, and analysis of their efficiency. Covers sorting algorithms, searching algorithms, dynamic programming and graph algorithms.

Lab: Application of algorithmic concepts through coding and problem-solving tasks.
Foundations of Computing Science Exploration of the theoretical foundations of computing, including discrete structures, formal languages, automata theory, and computability. Understanding the mathematical underpinnings of computation.
Ubiquitous Computing Study of computing in the context of everyday life, emphasizing the integration of computing into various aspects of the environment. Topics include mobile computing, sensor networks, and pervasive computing.
Artificial Intelligence Comprehensive overview of artificial intelligence concepts, including problem-solving, knowledge representation, machine learning, and deep learning. Introduction to AI applications and ethical considerations.
Machine Learning In-depth exploration of machine learning algorithms, techniques, and applications. Focus on supervised and unsupervised learning, feature engineering, and model evaluation.
Switching Circuits and Logic Design Understanding the design and analysis of digital circuits, including logic gates, sequential circuits, and combinational circuits. Covers switching theory and its application in computer systems.
Computer Architecture and Operating System Computer Architecture: Exploration of computer organization and architecture, including CPU design, memory systems, and input/output structures. Study of performance optimization and parallel processing.

Operating System: In-depth analysis of operating system concepts, process management, memory management, file systems, and system security. Practical exposure to designing and implementing basic operating system functionalities.
